Polaron
=======

Polaron is a sequencer/synthesizer/drummachine project based on the Teensy 3.6 plattfrom.
Warning: This project is work in progress and not working correctly yet.

Development Setup
=======
Arduino 1.8.6 / Teensyduino 1.43
